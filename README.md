# Django Chunked Upload Demo （Django 大文件分片断点续传 Demo）

非 Websocket 大文件分片断点续传

## 使用方法

```shell
pipenv install
pipenv shell

python manage.py migrate
python manage.py runserver
```

Visit [http://127.0.0.1:8000](http://127.0.0.1:8000)
